# 🚀 Ultimate AI Data Cleaner

## Περιγραφή
Ένα εργαλείο βασισμένο σε Python που χρησιμοποιεί Τεχνητή Νοημοσύνη (OpenAI GPT-4) για τον αυτόματο καθαρισμό και την τυποποίηση ακατάστατων (dirty) δεδομένων. 

## Key Features
- **Smart Name Formatting:** Αυτόματη διόρθωση τόνων, ορθογραφίας και Proper Case στα ονόματα.
- **Email Normalization:** Μετατροπή σε lowercase και αφαίρεση περιττών χαρακτήρων.
- **Phone Standardization:** Εξαγωγή καθαρών 10ψήφιων αριθμών, αφαιρώντας προθέματα και σύμβολα.
- **Batch Processing:** Δυνατότητα επεξεργασίας ολόκληρων αρχείων Excel/CSV.

## Τεχνολογίες
- **Python** (Pandas για τη διαχείριση των δεδομένων)
- **Streamlit** (για το Web Interface)
- **OpenAI API** (για το logic του καθαρισμού)
